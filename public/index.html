API Help

Before using this API you need to register as an user.

Use the below given url to register.

POST - https://enigmatic-coast-59622.herokuapp.com/users

You need to pass a JSON object as request body consisting of your email and password. For example -

{ "email": "your_email", "password": "your_password" }

API Routes

1. Login as user

POST - https://enigmatic-coast-59622.herokuapp.com/users/login

Request body - JSON Object e.g., { "email": "your_email", "password": "your_password" }

The generated response contains x-auth header which consists of a value. You need to use this value for subsequent requests.


2. Get all todos

GET - https://enigmatic-coast-59622.herokuapp.com/todo

Request header - Key-value pair e.g., {x-auth: "x-auth value"}


3. Add a todo

POST - https://enigmatic-coast-59622.herokuapp.com/todo

Request body - JSON Object e.g., { "text": "your_todo_text" }

Request header - Key-value pair e.g., {x-auth: "x-auth value"}


4. Get a todo by id

GET - https://enigmatic-coast-59622.herokuapp.com/todo/{id_of_todo}

Request header - Key-value pair e.g., {x-auth: "x-auth value"}


5. Update a todo by id

PATCH - https://enigmatic-coast-59622.herokuapp.com/todo/{id_of_todo}

Request header - Key-value pair e.g., {x-auth: "x-auth value"}


6. DELETE a todo by id

DELETE - https://enigmatic-coast-59622.herokuapp.com/todo/{id_of_todo}

Request header - Key-value pair e.g., {x-auth: "x-auth value"}


7. Logout

DELETE - https://enigmatic-coast-59622.herokuapp.com/users/me/token

Request header - Key-value pair e.g., {x-auth: "x-auth value"}
